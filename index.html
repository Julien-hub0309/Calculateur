Mini‑cours : Calculs sinusoïdaux

1. Définition d’une sinusoïde

Un signal sinusoïdal est une fonction périodique décrite par son amplitude, sa pulsation et sa phase :

u(t) = Umax · sin(ω · t + φ)
  • Umax : amplitude maximale (valeur crête)
  • ω : pulsation (rad/s)
  • φ : phase initiale (rad)
  • t : temps (s)

2. Relations fondamentales

  • Pulsation ↔ Fréquence
    ω = 2π · f  ⇔  f = ω / (2π)
  • Période
    T = 1 / f
  • Valeur efficace (RMS)
    Ueff = Umax / √2

La valeur efficace correspond au niveau DC équivalent pour la même puissance dissipée sur une résistance.

3. Caractéristiques graphiques

  • Amplitude : la courbe oscille entre −Umax et +Umax.
  • Période T : un cycle complet de la sinusoïde.
  • Phase φ : décale la courbe dans le temps (déphasage).

4. Exemple chiffré

Avec Umax = 10 V, ω = 2π · 50 rad/s, φ = 0 :

Fréquence
f = 50 Hz
Période
T = 0,02 s
Valeur efficace
Ueff ≈ 7,07 V
Signal
u(t) = 10 · sin(2π · 50 · t)

5. Applications

  • Électricité : tension et courant alternatifs (AC).
  • Audio : signaux sonores et synthèse.
  • Télécoms : modulation et transmission.
  • Physique : oscillations mécaniques, ondes.
